# My GitHub Repository

This repository contains two separate projects from my coursework:

## Projects

### 1. Blueprint Project (Java)
- **Folder:** `Blueprint/`
- **Description:** A Java project simulating an Apple AppStore-like blueprint system. Includes classes for `Unit`, `Floor`, and `Blueprint`, with JUnit tests to verify functionality.
- **Purpose:** Demonstrates object-oriented programming concepts, composition, and handling collections using primitive arrays.

### 2. Employee Management Project (Bash / C)
- **Folder:** `EmployeeManagement/`
- **Description:** A Bash and C project for managing employee data, including adding, updating, and listing employee information.
- **Purpose:** Practice file handling, arrays, loops, and conditional logic in Bash and C.
